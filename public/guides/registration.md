# 상품 등록 및 상태 변경 가이드

상품 등록부터 판매 완료까지, 끼리플리마켓에서 상품의 상태가 어떻게 변경되는지 상세히 안내해 드립니다.

## 1. 상품 등록 시 선택 사항
상품을 처음 등록할 때 거래 방식을 **일반 거래**와 <span class="inline-block text-[10px] px-2 py-0.5 font-bold rounded shadow-sm bg-orange-50 text-orange-600 border border-orange-100 align-middle -mt-0.5">경매</span> 중 하나로 선택할 수 있습니다. 
* **일반 거래**: 정해진 가격에 먼저 연락이 온 사람과 자유롭게 거래를 진행하는 방식입니다.
* **경매 거래**: 시작가와 최소 입찰 단위를 설정해 일정 기간 동안 입찰을 받는 방식입니다.

## 2. 상태값 변경 기준
끼리플리마켓의 상품은 상황에 따라 상태값을 가집니다. 각 상태의 의미와 변경 조건을 확인하세요.

### 판매중 (SALE)
* **정의**: 최초 상품 등록 시 자동으로 부여되는 상태입니다. 
* **설명**: 현재 거래를 희망하는 누구나 언제든 연락하거나 입찰할 수 있는 가장 기본적인 상태입니다. 

### <span class="inline-block text-xs px-2 py-0.5 font-bold rounded shadow-sm bg-teal-50 text-teal-600 border border-teal-100 align-middle">예약중</span>
* **정의**: 특정 구매자와 거래가 1차적으로 성사되었음을 의미합니다.
* **[일반 거래]**: 판매자가 특정 구매자와 채팅을 통해 거래를 약속한 뒤, 직접 상태를 예약중으로 변경할 수 있습니다.
* **[경매 거래]**: 설정된 시간이 다 되어 경매가 종료되거나, 판매자가 조기 낙찰을 눌렀을 때 **최고 입찰자가 있다면 자동으로 예약중으로 변경**됩니다.

### <span class="inline-block text-xs px-2 py-0.5 font-bold rounded shadow-sm bg-gray-100 text-gray-500 align-middle">판매완료</span>
* **정의**: 물건을 건네주고 거래가 최종적으로 확정된 상태입니다.
* **방법**: 판매자가 최종적으로 '판매 완료' 버튼을 누르면 상태가 변경됩니다.
* **매우 중요한 주의사항**: 판매완료 처리가 되면 시스템 내에 공식적인 **'거래 내역(Trade)'** 데이터가 생성됩니다. **이후에는 어떤 경우라도 다시 판매중이나 예약중 상태로 되돌릴 수 없습니다.** 꼭 구매자에게 물건을 건네준 뒤 확정해 주세요!

## 3. 상품 숨기기
* 잠시 판매를 보류하고 싶거나, 내 프로필에서 다른 사람들에게 보이지 않게 하고 싶다면 상품 상세 페이지에서 **'숨기기'** 기능을 사용할 수 있습니다.
* 숨겨진 상품은 피드나 검색에 노출되지 않으며, 원할 때 언제든 다시 보이게 해제할 수 있습니다.
