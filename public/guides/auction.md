# 🎈 경매 (Auction) 시스템 상세 가이드

끼리플리마켓만의 특별하고 재미있는 **'경매' 시스템**의 상세한 룰과 알고리즘을 안내해 드립니다.

## 1. 입찰 기본 규칙
* **최소 입찰 단위**: 경매 등록 시 설정된 최소 금액(기본 1,000원) 단위로만 입찰이 가능합니다.
* **최고 한도액**: 과열 방지를 위해 한 번에 10억 원을 초과하여 입찰할 수는 없습니다.
* **본인 입찰 금지**: 당연하지만, 판매자 본인이 올린 상품에는 직접 입찰할 수 없습니다.

## 2. 🍿 팝콘 타임 (마감 임박 자동 연장)
* 치열한 눈치싸움을 방지하고 모두에게 공정한 기회를 제공하기 위해 **'팝콘 타임'** 시스템이 적용되어 있습니다.
* 경매 마감 시간 **5분 이내**에 새로운 입찰이 발생할 경우, 경매 마감 시간이 **자동으로 5분 연장**됩니다. (누군가 마감 1분 전에 입찰하면, 마감 시간은 원래보다 4분 뒤로 늘어납니다!)

## 3. 판매자의 권한 통제
판매자는 경매를 원활하게 이끌기 위해 특별한 권한들을 가집니다.

### ⏱️ 조기 낙찰
* 설정한 경매 마감 시간이 되기 전이라도, 현재 입찰가와 입찰자가 마음에 든다면 판매자는 **'조기 낙찰'**을 눌러 경매를 즉시 종료할 수 있습니다. 
* 조기 낙찰 시 최고 입찰자가 낙찰자로 선정되며 상품은 '예약중'으로 변경됩니다.

### 🚫 최고 입찰 취소 (악성 입찰 방지)
* 1위 입찰자가 장난으로 입찰했거나 정상적인 거래가 불가능하다고 판단되는 경우, 판매자는 해당 입찰을 **'취소'**할 수 있습니다.
* 취소 시 1위 입찰 내역은 삭제되며, 2위 입찰자가 1위로 자동 승격되어 입찰가도 차순위 금액으로 롤백됩니다.

### 🔄 조기 낙찰 취소
* 조기 낙찰을 눌러 예약중 상태가 되었으나 실제 거래가 파기된 경우, 조기 낙찰을 취소할 수 있습니다.
* 이 경우 경매는 다시 '판매중' 상태로 돌아가며, 만약 원래의 경매 마감 시간이 이미 지났다면 시스템이 **자동으로 마감 시간을 24시간 연장**하여 경매를 재개시킵니다.

## 4. 🏁 경매 자동 종료 알고리즘
* 설정된 마감 시간이 지나면 백그라운드 시스템이 60초 주기로 상태를 체크하여 **자동으로 경매를 마감** 처리합니다.
* **입찰자가 있는 경우**: 최고 입찰자에게 낙찰 축하 알림이 발송되며 상태가 '예약중'으로 변경됩니다.
* **입찰자가 없는 경우**: 아무도 입찰하지 않은 상태로 종료되면 상품 상태는 '판매중'으로 유지되며, 판매자는 기한을 다시 연장하여 경매를 재오픈할 수 있습니다.
